## 📌 작성 규칙
1. **디렉토리와 파일명은 영어**로 작성합니다.
2. 학습한 내용을 **Markdown(.md)** 파일로 기록합니다.
3. **짧아도 매일** commit 합니다.
4. 코드 예시는 반드시 **코드 블록(\`\`\`)**을 사용합니다.
5. 링크와 참고 자료는 **즉시 첨부**하여 놓치지 않도록 합니다.

## 📝 Commit Convention
- `ADD` : 새 내용 추가  
- `FIX` : 오타 및 오류 수정  
- `UPDATE` : 기존 내용 보강  
- `REMOVE` : 불필요한 파일/내용 삭제
